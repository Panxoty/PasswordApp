- **Android_WS_Tarea**: Cómo conectar con un Servicio Web remoto y parsear datos en _JSON_
- **GuiaGasolineras2016**: Cómo trabajar con mapas, ubicaciones y el GPS. Proyecto adaptado a Android Studio
